## Download

[Go to download application](https://github.com/nonce-community/nonce-work-tools/releases)



## Development

### Dependencies

- [nodejs & npm](https://www.npmjs.com/get-npm)

### Install

```bash
cd /path/to/nonce-work-tools-project
npm install
```

### Run

```bash
npm run start
```

### Test

```bash
npm run test
```

### Build

```bash
npm run build
```
